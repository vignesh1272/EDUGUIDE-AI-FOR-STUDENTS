Overview

EduGuide AI uses two machine learning models to help students choose the right academic path and find suitable institutions.

🏫

Model 1: HSC School Recommendation

Inputs: Marks in Maths, Science, Social, English (35–100) and interest area.

Process: A Decision Tree classifier predicts the recommended group (e.g., Maths Biology, Commerce, Arts, Computer Science). It then filters schools that offer that group and have minimum mark requirements met.

Output: The predicted group and a list of eligible schools with details.

🎓

Model 2: College Recommendation

Inputs: Gender, 12th stream, percentage, location, parent's occupation, career interest.

Process: A Decision Tree classifier predicts a preferred degree. It then shows colleges offering that degree.

Output: The predicted degree and a list of recommended colleges.

🛠️

Technology Stack

  • Backend: Flask (Python)
  • Database: MySQL
  • Machine Learning: scikit-learn
  • Frontend: HTML5, CSS3 (glass morphism), JavaScript
